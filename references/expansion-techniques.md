# Expansion Techniques

Use these when the option space feels narrow, obvious, repetitive, or over-anchored. They are source material for mapper lenses, expansion passes, and workshop-like brainstorming.

## Divergent Then Convergent

Generate options before judging them. Keep exploration and selection as separate phases to reduce premature convergence.

## SCAMPER

Transform an existing idea:

- Substitute: What component, audience, medium, or constraint could be swapped?
- Combine: What two ideas could become stronger together?
- Adapt: What pattern from another use case could fit here?
- Modify: What could be enlarged, reduced, intensified, softened, or reshaped?
- Put to another use: Where else could this idea create value?
- Eliminate: What can be removed while preserving the core value?
- Reverse: What if the order, relationship, or assumption were inverted?

## Analogy And Far Transfer

Ask what the problem resembles in another domain, then import useful patterns.

- Biology.
- Games.
- Architecture.
- Parenting.
- Logistics.
- Education.
- Performance.

## Morphological Analysis

Break the problem into dimensions, list possible values for each dimension, then recombine values into new options.

Useful dimensions often include:

- Audience.
- Format.
- Scale.
- Tone.
- Cost.
- Timing.
- Channel.
- Success metric.

## Constraint Manipulation

Change the constraint set to reveal new possibilities:

- Remove a constraint.
- Exaggerate a constraint.
- Invert a constraint.
- Add a severe resource limit.
- Assume an abundant resource.

## Nominal Brainstorming

Generate ideas independently before combining them. This reduces anchoring, production blocking, and early dominance by one framing.

## Oblique-Style Provocations

Use provocations as controlled randomness, not as the main workflow.

### Invert

- What if the opposite were true?
- What would the anti-solution look like?
- What if the current weakness became the center?

### Remove

- What assumption can be deleted?
- What if the main resource disappeared?
- What if no one could coordinate?

### Exaggerate

- What if this had to be ten times bigger?
- What if it had to be ten times smaller?
- What if it had to happen in one day?

### Change Medium

- What would this look like as a game?
- What would this look like as a map?
- What would this look like as a ritual?
- What would this look like as an API?
- What would this look like as a workshop?
- What would this look like as a story?

### Use The Mistake

- What apparent flaw could become the core?
- What would be interesting if the failure were unavoidable?

### Shift Audience

- What would a child notice?
- What would a skeptic reject?
- What would an operator need?
- What would an artist amplify?
- What would a CFO question?
- What would a future user thank us for?
